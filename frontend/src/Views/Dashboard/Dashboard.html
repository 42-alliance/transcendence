ft_transcendence

Welcome to ft_transcendence, the final project of the Common Core at 42 School!

This project is a web application that brings together everything you've learned during your journey at 42. It challenges you to create a fully functional platform with real-time features, user authentication, and a seamless user experience.

In ft_transcendence, you'll build a website for playing Pong, complete with user profiles, matchmaking, chat functionality, and more. It's an opportunity to showcase your skills in backend development, frontend design, database management, and teamwork.

Are you ready to transcend your limits and create something extraordinary? Let's dive in!
